ARCHITECTURE

5 Layer Architecture

从基础到接口，依赖方向严格单向：上层依赖下层，下层不感知上层。

L0
LAYER 0

Foundation Layer

统一声明层 · 反射解析器 · 存储抽象 · 模型工厂 · 纯工具函数

L1
LAYER 1

Agent Core

中间件链编排 · 工具管线 · 知识包 · 任务委派 · 用户画像

L2
LAYER 2

Security Layer

隔离环境 · 文件安全 · 工具授权

L3
LAYER 3

Integration Layer

运行时执行 · 可观测性 · 外部工具 · 扩展工具集

L4
LAYER 4

Interface Layer

统一入口 · IM 通道 · Web 前端
