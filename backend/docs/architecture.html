Auth System

JWT + Provider + Repository 三层认证。支持本地邮箱密码登录，Token 版本轮换，CSRF 双重提交 Cookie。

Middleware Stack

CORS → Auth Middleware → CSRF Middleware 三级管道。全局 JWT 校验，请求级用户上下文注入。

routers/ deps.py

02
LAYER 02

Harness

Agent 运行时核心 — LangGraph Agent 工厂、18 层中间件链、工具装配、沙箱执行、记忆系统

🤖

Agent Factory

两层工厂模式：SDK 层 factory.py 纯参数驱动，应用层 lead_agent/agent.py 配置驱动。RuntimeFeatures 声明式特性开关。

factory.py lead_agent/ features.py
🔗

Middleware Chain

18 个有序中间件：从 ThreadData 初始化到 LoopDetection 检测，覆盖安全、性能、审计、上下文管理的全链路。

middlewares/ 18 components
🧰

Tool Assembly

四源工具装配管道：Config 定义 → MCP 扩展 → 内置工具 → 子代理。支持 Skill 管理工具的延迟加载。

tools/tools.py skills/
🧠

Memory System

四层架构：Prompt 注入格式化 → JSON 文件存储 → LLM 智能提取 → 防抖更新队列。支持 per-user 隔离。

memory/ 4 layers
📦

Sandbox

抽象沙箱接口，三种实现：Local（路径映射）、Docker（容器隔离）、AIO。支持 PathMapping 双向转换、只读挂载、输出脱敏。

sandbox/ local/
⚙️

Runtime

LangGraph 兼容运行时：RunManager 管理执行、Checkpointer 持久化状态、StreamBridge 流式通信、Store 抽象存储。

runtime/ runs/manager.py
