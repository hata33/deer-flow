## 二、Mini Agent 项目实现

### 2.1 项目概述

基于 DeerFlow 架构设计，创建了简化版 AI 代理系统。

**项目位置**: `/data/mini-agent-project/`

### 2.3 核心功能

**配置系统**
- Pydantic 模型验证
- YAML 配置文件
- 环境变量解析（`$VAR_NAME` 语法）
- 单例模式 + mtime 热更新

**沙箱系统**
- 抽象接口设计
- 本地文件系统实现
- 虚拟路径映射

**工具系统**
- 装饰器注册机制
- 4个内置工具（bash, read_file, write_file, list_dir）
- LangChain StructuredTool 集成

**代理引擎**
- LangChain 模型集成
- 工具调用支持
- 中间件链模式
- 状态管理

## 三、架构设计亮点

## 四、使用说明

### 4.1 快速启动

```bash
cd /data/mini-agent-project

# 安装依赖
pip install -r requirements.txt

# 配置 API 密钥
cp config.example.yaml config.yaml
# 编辑 config.yaml 设置 API 密钥

# 运行
python main.py
```

### 4.2 测试

```bash
pytest tests/ -v
```
