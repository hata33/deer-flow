🤖

agents/ - 代理引擎

基于 LangGraph StateGraph 的主代理实现，包含完整的 LLM 调用循环和工具执行管线

LeadAgent StateGraph Middleware Chain
🧠

memory/ - 记忆系统

四层架构：队列层(去重/优先级) → 存储层(JSON/mtime) → 注入层(上下文) → 使用层(事实提取)

4-Layer Decay Priority Queue
🛠️

tools/ - 工具系统

优先级装配管线：Config → MCP → Builtin → SubAgent，支持动态工具注册与权限控制

Priority Pipeline MCP Protocol Guardrails
📦

skills/ - 技能系统

基于 SKILL.md 的声明式插件机制，支持 Frontmatter 校验、安全扫描和动态加载

SKILL.md Hot Reload Security Scan
🔒

sandbox/ - 沙箱执行

双模式执行：本地沙箱(路径映射+输出遮蔽) 和 Docker 容器(完全隔离)，虚拟文件系统

Local Docker Virtual FS
🔄

subagents/ - 子代理

并发任务委派系统(最大3个)，双线程池架构(调度器+执行器)，事件驱动生命周期

Concurrent ≤3 ThreadPool Event-Driven
🌊
