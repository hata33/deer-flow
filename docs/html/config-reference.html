全局配置

必填

顶层字段

config.yaml
字段类型默认值说明
config_versionint9配置文件版本号，用于迁移兼容
log_levelstring"info"日志级别：debug / info / warning / error
namestring"DeerFlow"应用名称，显示在 UI 标题
token_usageobject{enabled: true}Token 用量统计开关
🤖

models — 模型配置

必填

模型注册

models:数组，可注册多个模型
字段类型必填说明
namestring必填模型内部标识符，全局唯一
display_namestring可选用户界面显示名称
usestring必填Python 类路径，如 langchain_openai:ChatOpenAI
modelstring必填模型参数，如 gpt-4 / claude-sonnet-4-20250514
api_keystring必填API 密钥，支持 $ENV_VAR 环境变量引用
base_urlstring可选自定义 API 端点（Ollama/vLLM 需要）
temperaturefloat可选生成温度 0.0-2.0
max_tokensint可选最大输出 Token 数
request_timeoutfloat600.0请求超时（秒）
max_retriesint2最大重试次数
supports_visionboolfalse是否支持图像输入
supports_thinkingboolfalse是否支持深度思考模式

已支持供应商

use 字段可选的 Python 类路径
供应商use 值示例 model

tools — 工具配置

可选

工具注册

tools:数组，每个工具独立配置
字段类型必填说明
namestring必填工具唯一名称，LLM 可见
groupstring必填工具分组：web / file:read / file:write / bash
usestring必填Python 类路径，如 deerflow.community.ddg_search.tools:web_search_tool
api_keystring可选工具专用 API Key，支持 $ENV_VAR

工具分组定义

tool_groups:
分组名权限级别典型工具
web低风险web_search, web_fetch — 网络访问
file:read低风险read_file, list_directory — 只读文件操作
file:write中风险write_file — 写文件操作
bash高风险bash — 终端命令执行
🔄

subagents — 子代理配置

可选

全局设置

subagents:
字段类型默认值说明
timeout_secondsint900子代理全局超时（秒）

内置子代理覆盖

subagents.agents:general-purpose / bash
字段类型说明
timeout_secondsint该子代理超时覆盖
max_turnsint最大对话轮数
modelstring使用的模型 name（覆盖默认模型）
skillsstring[]技能白名单，空数组 = 无技能

自定义子代理

subagents.custom_agents:自定义 key 即代理名
字段类型必填说明
descriptionstring必填子代理描述，LLM 可见（用于任务分配决策）
system_promptstring必填自定义系统提示词
toolsstring[]可选可用工具列表
skillsstring[]可选可用技能白名单
max_turnsint80最大对话轮数
timeout_secondsint600超时时间

sandbox — 沙箱配置

可选

沙箱设置

sandbox:
字段类型默认值说明
usestringdeerflow.sandbox.local:LocalSandboxProvider沙箱提供者类路径。可选 Local / Docker
allow_host_bashboolfalse是否允许直接在宿主机执行 bash（跳过沙箱）
config.yaml
# 本地沙箱 (默认)
sandbox:
  use: deerflow.sandbox.local:LocalSandboxProvider

# Docker 沙箱 (生产推荐)
sandbox:
  use: deerflow.sandbox.docker:DockerSandboxProvider
🧠

memory — 记忆配置

可选

记忆系统设置

memory:
字段类型默认值说明
enabledbooltrue记忆系统总开关
injection_enabledbooltrue是否注入到系统 Prompt
storage_pathstring"" (按用户)存储路径，空则按用户隔离；绝对路径=共享
storage_classstring(内置)存储提供者类路径，可自定义
debounce_secondsint30更新去抖动间隔（秒）
model_namestringnull事实提取使用的模型名，null 用默认模型
max_factsint100最大事实数量
fact_confidence_thresholdfloat0.7事实置信度阈值 0.0-1.0
max_injection_tokensint2000注入 Prompt 的最大 Token 数

guardrails — 安全策略

高级

Guardrail 配置

guardrails:
字段类型默认值说明
enabledboolfalse安全策略总开关
provider.usestring—Guardrail 提供者类路径
provider.configobject—提供者专属配置，如 denied_tools 列表
config.yaml
guardrails:
  enabled: true
  provider:
    use: deerflow.guardrails.builtin:AllowlistProvider
    config:
      denied_tools: ["bash"]  # 禁止的工具列表

extensions_config.json — MCP 配置

可选

MCP Server 字段

extensions_config.json → mcpServers.{name}
字段类型必填说明
enabledbool必填是否启用
typestring必填传输类型：stdio / sse / http
commandstringstdio 必填可执行命令（type=stdio）
argsstring[]可选命令参数
urlstringsse/http 必填远程服务 URL
envobject可选环境变量注入
descriptionstring可选服务描述，LLM 可见
