Backend Modules Detail

🚪

app/main.py - 应用入口

FastAPI 应用实例创建，生命周期管理 (lifespan)，中间件注册，路由挂载，CORS 配置

FastAPI Lifespan Middleware
📡

channels/ - IM 通道

多平台 IM 集成层。每个平台独立模块，由 ChannelManager 统一管理，通过 MessageBus 路由消息

Discord Slack Telegram 微信/企微 钉钉 飞书
🔀

gateway/routers/ - API 路由

RESTful API 端点集合：代理管理、会话线程、技能加载、记忆管理、文件上传、认证授权

agents threads skills memory artifacts
🔐

gateway/auth/ - 认证系统

JWT Token 签发/验证，CSRF 保护，权限校验中间件，支持多用户会话管理

JWT CSRF OAuth

agents/ - 代理引擎

系统最核心模块。LeadAgent 基于 LangGraph StateGraph 构建，管理完整的 LLM 调用循环。包含约 20 个中间件形成处理链，覆盖初始化、模型调用包装、后处理、工具调用包装四个阶段

LeadAgent StateGraph ~20 Middlewares LLM Loop
⚙️

runtime/ - 运行时管理

RunManager 负责创建和跟踪每次运行；StreamBridge 通过有界缓冲区将后端事件桥接到前端 SSE；RunJournal 记录 LangChain 回调事件；EventStore 存储运行元数据

RunManager StreamBridge RunJournal EventStore
🛠️

tools/ - 工具系统

优先级装配管线：配置工具(config.yaml) → MCP 协议工具 → 内置工具(bash/read_file/write_file/list_directory) → 子代理工具(task)。支持动态注册、权限控制、命令审计

Priority Pipeline Bash Tool File Tools Guardrails
📦

skills/ - 技能系统

基于 SKILL.md 的声明式插件。解析流程：SKILL.md 扫描 → Frontmatter 校验 → 安全扫描 → 包安装 → 工具注入 → 动态加载。支持热重载和能力扩展

SKILL.md Frontmatter Security Scan Hot Reload
🔒

sandbox/ - 沙箱引擎

双模式安全执行：Local Provider (路径映射 + 输出遮蔽) 和 Docker Provider (容器隔离)。虚拟文件系统 /mnt/ 前缀，防止路径遍历攻击

Local Sandbox Docker Container Virtual FS Path Mapping
🔄

subagents/ - 子代理

并发任务委派系统。最大 3 个子代理并行执行，采用双线程池架构(调度器线程池 + 执行线程池)，事件驱动生命周期：started → running → completed/failed

Concurrent ≤3 Dual ThreadPool Event-Driven task()
🔌

mcp/ - MCP 协议

Model Context Protocol 客户端实现。管理外部 MCP Server 连接，自动发现可用工具，支持工具调用、资源访问等协议功能

MCP Client Tool Discovery Server Management
💾

persistence/ - 持久化

SQLAlchemy ORM 抽象层，支持多数据库后端(PostgreSQL/SQLite)。管理 Thread(会话)、Run(运行)、Memory(记忆) 等核心实体

SQLAlchemy PostgreSQL Multi-Backend
💬
