## 核心数据流

```
 用户消息
    │
    ▼
 Nginx (:2026) ──► Gateway FastAPI (:8001)
    │                       │
    │  CORS → CSRF → JWT → 权限
    │                       │
    │                       ▼
    │                  RunManager
    │                       │
    │                       ▼
    │                  run_agent()
    │                       │
    │                       ▼
    │                  Lead Agent
    │                       │
    │            ┌──────────┼──────────┐
    │            ▼          ▼          ▼
    │         中间件链    LLM 调用   工具执行
    │            │          │          │
    │            │          ▼          ▼
    │            │      Models     Sandbox
    │            │      反射工厂   虚拟路径
    │            │                     │
    │            │                Guardrails
    │            │                策略授权
    │            │
    │            ├── Skills  ──► SKILL.md 按需加载
    │            ├── Memory  ──► 用户画像 + facts
    │            └── Subagents ──► task() 并行执行
    │
    ▼
 Frontend SSE ◄──── StreamBridge ◄──── RunJournal
 三源合并渲染        有界缓冲          LangChain 回调
```

## 中间件链执行顺序

```
 请求进入
    │
    ▼
 before_agent
    ├── ThreadDataMiddleware          线程目录管理
    ├── UploadsMiddleware             上传文件注入
    ├── SandboxMiddleware             沙箱环境初始化
    ├── DanglingToolCallMiddleware    悬挂调用修补
    ├── DynamicContextMiddleware      记忆/日期动态注入 ◄── Memory
    └── DeferredToolFilterMiddleware  延迟工具过滤
    │
    ▼
 wrap_model_call
    └── LLMErrorHandlingMiddleware    错误重试 + 熔断
    │
    ▼
 LLM 调用 ──────────────────────────► Models (BaseChatModel)
    │
    ▼
 after_model
    ├── TokenUsageMiddleware          Token 统计 + 步骤归属
    ├── TitleMiddleware               自动标题生成
    ├── LoopDetectionMiddleware       循环检测 + 强制停止
    ├── TodoMiddleware                任务追踪
    └── SubagentLimitMiddleware       子代理并发限制
    │
    ▼
 wrap_tool_call
    ├── GuardrailMiddleware           安全护栏 ◄── Guardrails
    ├── SandboxAuditMiddleware        Bash 命令审计
    ├── ToolErrorHandlingMiddleware   工具异常转 ToolMessage
    └── ClarificationMiddleware       澄清请求拦截
    │
    ▼
 after_agent
    └── MemoryMiddleware              记忆更新排队 (30s 防抖) ──► Memory
    │
    ▼
 返回结果
```
