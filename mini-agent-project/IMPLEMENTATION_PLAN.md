# Mini Agent - 简化版 AI 代理系统实现计划

## 项目概述

基于 DeerFlow 架构设计，创建一个精简但功能完整的 AI 代理系统。

## 设计目标

- **简化但完整**：保留核心功能，移除过度设计的部分
- **易于理解**：清晰的代码结构和文档
- **可扩展**：模块化设计，便于添加新功能
- **生产就绪**：包含错误处理、日志、测试

## 核心功能

### 1. 配置系统
- Pydantic 配置模型
- YAML 配置文件
- 环境变量解析
- 热更新支持

### 2. 代理引擎
- LangGraph 状态机
- 工具调用
- 中间件支持
- 记忆管理

### 3. 工具系统
- 内置工具（bash, 文件操作）
- 自定义工具注册
- 工具参数验证

### 4. 沙箱系统
- 本地文件系统沙箱
- 虚拟路径映射
- 命令执行

### 5. 记忆系统
- 对话历史存储
- 上下文注入
- 摘要压缩

## 与 DeerFlow 的对比

| 功能 | DeerFlow | Mini Agent |
|------|----------|------------|
| 配置系统 | 18个配置文件 | 3个核心配置 |
| 中间件 | 12个 | 5个基础中间件 |
| 工具 | 内置+MCP+社区 | 内置+自定义 |
| 沙箱 | Local + Docker | Local |
| 记忆 | LLM驱动 | 简单存储 |
| 检查点 | 支持 | 可选 |
| 子代理 | 支持 | 暂不支持 |

## 技术栈

- **Python**: 3.12+
- **Pydantic**: 配置验证
- **LangGraph**: 状态机
- **LangChain**: LLM 集成
- **PyYAML**: 配置解析
