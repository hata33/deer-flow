系统提示词——动态模板组装、记忆注入、技能注入、子代理段落

---

## 一、入口函数

**文件**：`agents/lead_agent/prompt.py` → `apply_prompt_template(subagent_enabled, max_concurrent_subagents, agent_name, available_skills)`

返回完整的系统提示词字符串，由 `SYSTEM_PROMPT_TEMPLATE`（Python 多行字符串模板）通过 `.format()` 动态填充

---

## 二、模板结构

`SYSTEM_PROMPT_TEMPLATE` 包含以下 XML 标签段落，对应 `{placeholder}`：

```
<role>
  You are {agent_name}, an open-source super agent.
</role>

{soul}                    ← <soul> SOUL.md 人格描述 </soul>

{memory_context}          ← <memory> 记忆数据 </memory>

<thinking_style>          ← 思考风格指导（固定 + {subagent_thinking}）
</thinking_style>

<clarification_system>    ← 澄清工作流（固定，5 种澄清类型）
</clarification_system>

{skills_section}          ← <skill_system> 可用技能列表 </skill_system>

{deferred_tools_section}  ← <available-deferred-tools> 延迟工具名 </available-deferred-tools>

{subagent_section}        ← <subagent_system> 子代理编排规则 </subagent_system>

<working_directory>       ← 工作目录说明（固定 + {acp_section}）
</working_directory>

<response_style>          ← 响应风格（固定）
</response_style>

<citations>               ← 引用格式规则（固定）
</citations>

<critical_reminders>      ← 关键提醒（固定 + {subagent_reminder}）
</critical_reminders>

<current_date>            ← 当前日期（动态追加到末尾）
</current_date>
```

## 三、动态段落详解

### 3.1 soul——智能体人格

```python
def get_agent_soul(agent_name):
    soul = load_agent_soul(agent_name)
    if soul:
        return f"<soul>\n{soul}\n</soul>"
    return ""
```

从 `agents_config` 加载指定智能体的 `SOUL.md` 内容。未配置时为空字符串，不影响模板

### 3.2 memory_context——记忆注入

```python
def _get_memory_context(agent_name):
    config = get_memory_config()
    if not config.enabled or not config.injection_enabled:
        return ""

    memory_data = get_memory_data(agent_name)
    memory_content = format_memory_for_injection(memory_data, max_tokens=config.max_injection_tokens)

    if not memory_content.strip():
        return ""

    return f"<memory>\n{memory_content}\n</memory>"
```

注入时机：**agent 创建时**（构建提示词时），不是运行时

格式化规则（`format_memory_for_injection`）：
- User Context：workContext + personalContext + topOfMind
- History：recentMonths + earlierContext
- Facts：按 confidence 降序排列，受 `max_injection_tokens` 限制截断

记忆更新的延迟：本轮 MemoryMiddleware 排队 → 防抖计时器 → LLM 更新 → 写入 memory.json → **下轮** `_get_memory_context()` 才能读到

详见 [006-记忆系统.md](006-记忆系统.md)

### 3.3 skills_section——技能注入

```python
def get_skills_prompt_section(available_skills=None):
    skills = load_skills(enabled_only=True)
    if available_skills is not None:
        skills = [s for s in skills if s.name in available_skills]

    # 生成 XML 格式的技能列表
    skill_items = "\n".join(
        f"    <skill>\n"
        f"        <name>{skill.name}</name>\n"
        f"        <description>{skill.description}</description>\n"
        f"        <location>{skill.get_container_file_path(base_path)}</location>\n"
        f"    </skill>"
        for skill in skills
    )
    return f"<skill_system>...\n<available_skills>\n{skill_items}\n</available_skills>\n</skill_system>"
```

**Progressive Loading 模式**：提示词只告诉 agent 技能文件的路径，agent 通过 `read_file` 工具按需读取内容：

```
1. 用户问题匹配某个 skill 的 description
2. agent 调用 read_file(skill.location) 读取技能内容
3. 技能文件中包含引用的外部资源路径
4. 按需加载引用资源
5. 按技能指令执行
```

### 3.4 deferred_tools_section——延迟工具名

```python
def get_deferred_tools_prompt_section():
    if not config.tool_search.enabled:
        return ""
    registry = get_deferred_registry()
    if not registry:
        return ""
    names = "\n".join(e.name for e in registry.entries)
    return f"<available-deferred-tools>\n{names}\n</available-deferred-tools>"
```

只列出工具名（不包含 schema），让 agent 知道哪些工具可以通过 `tool_search` 加载

### 3.5 subagent_section——子代理编排

`_build_subagent_section(max_concurrent)` 动态生成，核心内容：

```
硬性约束：每次响应最多 N 个 task 调用（超出被系统丢弃）

编排策略：
  DECOMPOSE → 分解为并行子任务
  DELEGATE  → 启动 N 个子代理
  SYNTHESIZE → 收集结果综合回答

多批次执行（子任务 > N 时）：
  Turn 1: 启动前 N 个 → 等结果
  Turn 2: 启动下一批 → 等结果
  Final: 综合所有结果

不可分解的任务 → 直接执行，不用子代理
```

同时生成 `{subagent_reminder}` 和 `{subagent_thinking}` 两个辅助段落

### 3.6 acp_section——ACP 代理说明

```python
def _build_acp_section():
    agents = get_acp_agents()
    if not agents:
        return ""
    return "ACP agents run in independent workspace — NOT in /mnt/user-data/ ..."
```

只在配置了 ACP 代理时生成，说明工作空间隔离规则

## 四、模板外的追加

在模板 `.format()` 之后，额外追加当前日期：

```python
return prompt + f"\n<current_date>{datetime.now().strftime('%Y-%m-%d, %A')}</current_date>"
```

> 本文档：系统提示词由 `SYSTEM_PROMPT_TEMPLATE` 模板动态填充。soul 来自 SOUL.md 人格、memory 从 memory.json 注入（构建时非运行时）、skills 声明式引用（Progressive Loading）、subagent 含硬性并发限制、deferred_tools 只列名称。日期在最后追加
