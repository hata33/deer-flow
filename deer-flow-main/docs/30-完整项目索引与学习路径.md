## 推荐学习路径

### 路径一：快速上手（1-2小时）

**目标**：快速理解项目架构，能够运行系统

1. **00-全集总览与全仓库拓扑架构.md** → 了解项目整体结构
2. **08-前端架构总览.md** → 理解用户界面
3. **22-配置系统深度解析.md** → 学习配置方式
4. **29-部署与运维指南.md** → 完成本地部署

### 路径二：核心原理（4-6小时）

**目标**：深入理解AI代理核心机制

1. **16-代理系统深度解析.md** → 理解代理运行机制
2. **21-模型工厂与多模型支持系统.md** → 学习模型管理
3. **17-沙箱系统深度解析.md** → 理解代码执行隔离
4. **18-工具系统深度解析.md** → 学习工具加载机制
5. **24-记忆系统深度解析.md** → 理解长期记忆实现

### 路径三：扩展开发（6-8小时）

**目标**：掌握扩展系统开发技能

1. **19-MCP集成系统深度解析.md** → 学习外部工具集成
2. **20-技能系统深度解析.md** → 学习技能创建
3. **23-反射系统与动态模块加载.md** → 学习动态加载
4. **15-IM渠道集成系统.md** → 学习多平台集成
5. **28-API路由系统详解.md** → 学习后端API开发

### 路径四：全栈精通（10+小时）

**目标**：全面掌握系统设计与实现

**按顺序阅读所有文档**，每个文档都包含：
- 模块全局定位
- Mermaid架构图
- 核心源码解析
- 设计思想解读
- 可复用代码片段
- 踩坑提醒与优化建议

## 技术栈索引

### 后端技术

- **Python 3.12+**：主要开发语言
- **FastAPI**：Gateway API框架
- **LangGraph**：代理编排框架
- **Pydantic**：配置验证
- **LangChain**：LLM集成框架

### 前端技术

- **Next.js 16**：React框架
- **React 19**：UI库
- **TypeScript 5.8**：类型系统
- **Tailwind CSS 4**：样式框架
- **TanStack Query**：服务端状态管理

### 基础设施

- **Docker**：容器化部署
- **Nginx**：反向代理
- **PostgreSQL**（可选）：持久化存储
- **Redis**（可选）：缓存与队列
